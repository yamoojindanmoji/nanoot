## 공통 작업 규칙
- 작업 전 항상 관련 파일 먼저 파악
- 브랜치 머지 전 충돌 파일 보고 후 승인 받기
- 임의로 충돌 해결 금지
- 작업 완료 후 git push까지

## 작업 완료 후 필수 사항
- 작업 완료 후 반드시 로컬 환경에도 변경사항이 적용되었는지 확인해줘
- 로컬에서 npm run dev 실행해서 동작 확인 후 git push 해줘
- 브랜치 작업 시 git push origin [브랜치명] 해줘

## 디자인 가이드
UI 작업 시 항상 DESIGN_GUIDE.md 파일을 먼저 읽고
해당 디자인 시스템을 준수해서 작업해줘.
